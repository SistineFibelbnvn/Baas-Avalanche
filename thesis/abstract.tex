\chapter*{ABSTRACT}
\addcontentsline{toc}{chapter}{Abstract}

This thesis presents an in-depth study of Layer 1 Blockchain architecture on the Avalanche platform and the development of a comprehensive software system that enables users to create, operate, and monitor custom Blockchain networks (Subnets) through an intuitive web interface.

On the research front, the thesis analyzes the Snowball consensus protocol --- a probabilistic protocol based on repeated random sub-sampling --- and compares it with traditional BFT protocols across performance, fault tolerance, and decentralization criteria. Additionally, the thesis examines Avalanche's multi-chain architecture including the Primary Network (X-Chain, P-Chain, C-Chain) and the extensible Subnet model.

On the engineering front, the system is built with a Modular Monolith architecture comprising: (1) a Backend Orchestrator developed with NestJS integrating Docker API for validator node lifecycle management; (2) a Frontend Dashboard developed with Next.js featuring WebSocket for real-time monitoring; (3) a Monitoring Stack integrating Prometheus and Grafana. The system supports user authentication via JWT and Google OAuth with resource-ownership-based authorization.

Experimental results demonstrate that the system reduces Subnet deployment time from hours (manual CLI operations) to under 2 minutes through a web wizard interface, while providing a visual monitoring dashboard with sub-3-second update latency.

\vspace{0.5cm}
\noindent\textbf{Keywords:} Blockchain Layer 1, Avalanche Subnet, Blockchain-as-a-Service, Snowball consensus protocol, Docker container, real-time monitoring.
